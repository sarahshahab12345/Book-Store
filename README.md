# Book Store Project

## Technologies Used

- **Backend:** Node.js, Express
- **Frontend:** Vite, React, Material UI
- **Database:** MongoDB

## Functionality

This project implements a Book Store with CRUD (Create, Read, Update, Delete) operations for managing books.
